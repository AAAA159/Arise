## 📌 عن المشروع

هذا هو موقعي الشخصي كمطور ويب. صممته ليكون واجهتي الرقمية التي أعرض فيها مهاراتي، مشاريعي، وأستقبل طلبات العملاء عبر واتساب مباشرة.

**المميزات الرئيسية:**
- تصميم أسود وبنفسجي عصري
- خلفية Three.js ثلاثية الأبعاد بتتحرك مع الماوس
- معرض أعمال بتصميم Bento Grid
- نموذج تواصل مرتبط برقم واتساب مباشر
- متجاوب مع جميع الأجهزة (Responsive)
- شهادات عملاء للتأكيد على المصداقية
- تأثيرات حركية سلسة باستخدام GSAP

## 🚀 كيفية التشغيل

### الطريقة الأولى: تشغيل محلي
```bash
# 1. حمل الملف
git clone https://github.com/your-username/arise-portfolio.git

# 2. افتح الملف في المتصفح
cd arise-portfolio
open index.html
```

الطريقة الثانية: نشر على الإنترنت

✅ GitHub Pages (مجاني)

```bash
# 1. ارفع الملف على GitHub
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/your-username/arise-portfolio.git
git push -u origin main

# 2. فعل GitHub Pages من إعدادات المستودع
# Settings → Pages → Source: Deploy from a branch → Branch: main → Save
```

✅ Netlify (مجاني وسهل)

· اسحب ملف index.html وأفلته على app.netlify.com/drop
· خلال ثواني الموقع هيكون منشور برابط جاهز

✅ Vercel (مجاني)

```bash
npm i -g vercel
vercel
```
