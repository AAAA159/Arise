## 📌 عن المشروع

هذا هو موقعي الشخصي كمطور ويب. صممته ليكون واجهتي الرقمية التي أعرض فيها مهاراتي، مشاريعي، وأستقبل طلبات العملاء عبر واتساب مباشرة.

### ✨ المميزات الرئيسية:
* **تصميم عصري:** واجهة مستخدم باللونين الأسود والبنفسجي الداكن (Dark Mode).
* **خلفية ثلاثية الأبعاد:** تأثيرات تفاعلية باستخدام Three.js تتحرك مع حركة الماوس.
* **معرض أعمال احترافي:** تصميم بأسلوب الـ Bento Grid العصري لعرض المشاريع بشكل غير متماثل.
* **ربط مباشر بالواتساب:** نموذج تواصل ذكي يفتح محادثة واتساب مباشرة مع العميل.
* **تأثيرات حركية:** حركات سكرول سلسة وجذابة باستخدام مكتبة GSAP.
* **شهادات العملاء:** قسم خاص بآراء العملاء لتعزيز المصداقية وبناء الثقة.
* **متجاوب بالكامل:** متوافق تماماً مع جميع الشاشات (الهواتف، التابلت، والحواسيب).
